Simple Scientific Calculator

This Python application is a GUI-based scientific calculator built using Tkinter. It features an intuitive layout and supports both basic and advanced mathematical operations.

Key Features:

  • Basic Operations: Addition, subtraction, multiplication, and division.
  • Advanced Functions:
    • Trigonometric functions: sin, cos, tan, and their hyperbolic versions.
    • Logarithmic functions: log, ln.
    • Power functions: x², x³, ^ (custom exponentiation).
    • Constants: π (pi), e (Euler's number).
    • Others: Square root, reciprocal, rounding (floor, ceiling).
  • Error Handling: Displays "Error" for invalid operations or inputs.
  • Responsive UI: Buttons change color on hover for better user experience.

How It Works:

  1. User inputs an expression using buttons or the on-screen display.
  2. Operators like ^, π, and e are internally replaced with Python-compatible equivalents.
  3. Clicking "=" evaluates the expression and displays the result.
  4. Special buttons trigger advanced mathematical functions.

Usage:

Run the script in Python to launch the calculator GUI. Click buttons to perform operations and view results instantly.
